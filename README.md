# Mountain Shooter

> Projeto desenvolvido durante as aulas práticas da disciplina de Linguagem de Programação do curso de Engenharia de Software da UNINTER.

Mountain Shooter é um jogo arcade 2D desenvolvido com Python e Pygame. O jogador enfrenta ondas de inimigos em cenários de montanha, acumulando pontos enquanto tenta sobreviver até o fim de cada fase.

## Sobre o Projeto

O Mountain Shooter foi criado com o objetivo de aplicar na prática conceitos de programação orientada a objetos, manipulação de eventos, desenvolvimento de jogos e integração com banco de dados utilizando Python.

O jogo possui:

- 2 fases jogáveis;
- Modo cooperativo para 2 jogadores;
- Sistema de pontuação;
- Armazenamento de scores em banco de dados SQLite;
- Controle de inimigos e projéteis;
- Interface gráfica desenvolvida com Pygame.

## Tecnologias Utilizadas

- Python 3
- Pygame
- SQLite3
